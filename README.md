## 生成 APK
### 使用命令行：
你可以打开终端（在 Android Studio 中，你可以通过 View -> Tool Windows -> Terminal 打开），然后导航到你的项目根目录并运行以下命令：
```bash
./gradlew assembleRelease
```
这将为你的 release 构建变体生成 APK。
